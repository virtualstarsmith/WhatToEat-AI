# getPoi 云函数配置说明

## 环境变量配置

此云函数需要配置高德地图 API Key。

### 配置步骤

1. 访问 [高德地图开放平台](https://lbs.amap.com/)
2. 注册/登录账号
3. 创建应用，选择"Web服务"类型
4. 获取 Key

### 在微信开发者工具中设置

1. 打开**云开发控制台**
2. 进入**云函数** → **getPoi**
3. 点击**配置** → **环境变量**
4. 添加环境变量：
   - **键**: `AMAP_KEY`
   - **值**: 你的高德地图 Key

### 重新部署

设置环境变量后，需要重新上传并部署云函数：
- 右键点击 `cloudfunctions/getPoi` 文件夹
- 选择**上传并部署：云端安装依赖**
